The API I selected was a Covid-19 cases/deaths tracker for all countries. This is very useful for people who want to know how many cases there have been, and if the current amount of cases is spiking. It is useful to also compare the number of cases and deaths in different countries to see how the preventative measures compare.

The website is linked above!

I used fetch to get the data from the website as a JSON object, and then parsed the data to display it on my page. I also used AJAX to get the data through a request, then read the data as a JSON object, and parsed it to display it on the page.

This API is useful for anyone who wants to know the statistics about covid cases and deaths in any country. If someone wanted to see if there was a correlation between the amount of preventative measures taken in one country compared to another, they could use this API to check and compare the number of cases + deaths between countries.
